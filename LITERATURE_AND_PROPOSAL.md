## 1. 선행연구 리뷰 요약

### 1.1 문서 성격

논문 3장에 해당하는 **선행연구 리뷰**. 스켈레톤/루지/봅슬레이 경기력 결정 요인을 7개 범주로 분류하고, 각 요인별 선행연구 근거를 제시한 뒤 본 연구와 연결.

### 1.2 7대 요인 구조

| # | 요인 범주 | 핵심 선행연구 | 본 연구 적용 방향 |
|---|----------|-------------|-----------------|
| 1 | **신체 (키, 체중, 근력)** | Vracas: 총질량↑→중력가속↑→기록단축 / Colyer: 스프린트+점프+레그프레스 → 스타트 R²=0.86 | 키·체중을 질량·CDA 변인으로, 근력은 Start 기록에 내포 |
| 2 | **장비 (러너, 공기역학)** | Poirier F.A.S.T.: 록커↑→마찰 12%↓ / Vracas: CDA 최적화→0.2초+ 단축 | 장비 동일 조건 통제, 마찰은 환경 변인으로 간접 반영 |
| 3 | **트랙 기하학** | Vracas: 1D 운동방정식으로 하강시간 재현 / Moßner: 트랙 설계 시뮬레이션 | **단일 트랙(평창)** 고정, 트랙 간 비교는 차후 연구 |
| 4 | **환경·빙질·기상** | F.A.S.T.: 얼음온도→마찰 비선형 / 육상 연구: 공기밀도→기록 비선형 영향 | 공기밀도(기온·습도·기압) + 얼음온도 + 이슬점 |
| 5 | **스타트·주행 지표** | 0.1초 스타트 향상 → 0.15~0.2초 피니시 단축 (국제대회 반복 확인) | Start 기록(초)을 핵심 독립변인 |
| 6 | **심리·전술** | IAPPF: 심리 포함 시 예측력 크게 향상 | 본 연구에서는 제외 (통제되지 않은 외생 요인) |
| 7 | **통합 모델** | 신체+환경+스타트 통합 다변량 회귀 | 선행연구에 거의 없는 영역 → 본 연구의 차별성 |

### 1.3 주요 인용 연구 및 핵심 발견

#### Vracas et al. (2023) — Altenberg 트랙 시뮬레이션

- 질량, 스타트 속도, 공기저항(CDA), 아이스 마찰의 민감도 분석
- 총질량을 규정 한도까지 증가 + 스타트·자세 최적화 + 러너 마찰 최소화 → 기록 유의 단축
- 트랙 궤적을 파라미터화하여 1D 운동방정식으로 하강시간 재현 (±2% 이내 오차)
- high/mid/low line 3가지 궤적 가정 → 궤적에 따라 거리·원심력·마찰 손실 달라짐
- **스타트 속도 차이 0.1초 → 피니시 0.15~0.2초 단축**

#### Poirier (2011) — F.A.S.T. 3.2b 마찰 모델

- 러너-얼음 계면의 quasi-liquid layer 형성과 열전도, 압출유동(squeeze flow) 고려
- 러너 단면 반경 4.75→5.5mm → 마찰계수 약 0.5% 감소
- 록커 34→48m → 마찰계수 약 12% 감소 ("더 평평한 러너가 마찰을 줄인다")
- **얼음 온도가 마찰에 비선형 영향** — 너무 차갑거나 너무 따뜻하면 마찰 증가
- 최적 속도 구간: 얼음 표면 온도 -10~-20°C 근처

#### Colyer et al. (2017) — 엘리트 스켈레톤 스타트 성능

- 엘리트 24명 대상
- 15m 무저항 스프린트 + 수직점프 높이 + 레그프레스 파워 → 스타트 성능(15m 썰매 속도) 예측
- **회귀식 설명력 R²=0.86**
- 근력·파워를 직접 측정하지 않아도 스타트 기록에 내포된 결과로 취급 가능

#### Bayreuth 그룹 — 글라이드 모델

- 피니시 타임 = f(질량, drag area Ad, 마찰계수 μ)
- 항력식: F_D = 0.5ρA_d v²
- 국제대회 상위 선수 데이터로 규정상 질량 제한의 공정성 논의

#### Jiang et al. (2023) — 프리스타일 스키 딥 트랜스퍼 러닝

- Unity3D 시뮬레이션으로 가상 데이터 생성 후 실측 데이터와 결합
- 눈 마찰계수·풍속·기온·경사·선수 체격 → 런칭 속도 예측
- 환경·신체 변인 결합 예측모델의 실전 코칭 활용 가능성 입증

#### Qin et al. (2025) — 통합 예측모형(IAPPF)

- 생체신호 + 심리적 요인 + 환경·훈련 맥락 융합
- 단일 차원 모델보다 높은 예측력 확보
- 본 연구는 심리 제외, 신체+환경+주행변인 통합이라는 점에서 방향성 공유

### 1.4 선행연구가 제안하는 변인 구조

```
┌─ 신체 변인 ──────────────────────────────────────┐
│  키, 체중 → 총질량(중력가속) + CDA(공기저항)       │
│  근력·파워 → 스타트 기록에 내포                    │
├─ 환경 변인 ──────────────────────────────────────┤
│  공기밀도(기온·습도·기압) → 항력 F_D = 0.5ρv²CDA  │
│  얼음 온도 → 마찰계수 μ (비선형)                   │
│  이슬점 → 서리 발생 여부 → 추가 마찰 증가          │
├─ 주행 변인 ──────────────────────────────────────┤
│  스타트 기록(초) → 0.1초 향상 = 0.15~0.2초 단축   │
├──────────────────────────────────────────────────┤
│                    ↓                              │
│            Finish Time (종속변수)                  │
└──────────────────────────────────────────────────┘
```

### 1.5 논문에서 명시한 차별성

> "썰매 종목에서 선수 신체 특성, 빙질·기상 환경, 스타트 기록을 한꺼번에 독립변수로 설정하여 피니시 타임을 다변량 회귀로 예측하고, 그 결과를 신인·유소년 선수 대상 코스 전략 및 훈련 처방에 직접적으로 연결한 연구는 아직 매우 제한적이다."

### 2.2 연구 목적

- 신체적 특성(키, 몸무게) + 주행 환경 변인(얼음 온도, 기상 상태) + 스타트 기록 → **Finish 예측 범용 다중선형회귀 모델** 개발
- **핵심**: 특정 선수의 개인 주행 이력에 의존하지 않고 **신인/유스 선수도 예측 가능**한 범용 모델
- 맞춤형 훈련 전략 수립을 위한 객관적 코칭 지표 제공

### 2.3 배경 및 이론적 근거 (4가지)

| # | 근거 | 핵심 논리 |
|---|------|----------|
| 1 | 현장 데이터의 축적과 분석의 괴리 | 방대한 데이터가 축적되지만 체계적 분석 모델 부재, 0.01초 단위 초정밀 종목에서 지도자의 '감'에 의존 |
| 2 | 썰매 종목의 다변량 예측 연구 부재 | 육상·수영은 ML 기반 예측 연구 활발, 썰매는 국내외 극히 드뭄 |
| 3 | 주행 기록 결정 요인의 역학적 접근 | Finish = f(중력하중+스타트, 공기저항, 빙면마찰, 장비) → 다중선형회귀가 이론적으로 가장 타당 |
| 4 | 데이터 기반 코칭 패러다임 전환 | "체중 1kg 증가 시 / 스타트 0.05초 단축 시 Finish 몇 초 단축?" 정량적 수치 제공 |

### 2.4 변인 설계

```
독립변수 (X)                                 종속변수 (Y)
┌─ ① 신체 변인 ─────────────────┐
│    키(cm), 몸무게(kg)          │
├─ ② 환경 변인 ─────────────────┤
│    공기밀도(기온/습도/기압 파생) │──────→  Finish Time(초)
│    이슬점 온도(서리 판별)       │

├─ ③ 주행 변인 ─────────────────┤
│    목표/예상 스타트 기록(초)     │

### 2.5 연구 방법

| 단계 | 내용 |
|------|------|
| **대상** | 국내 스켈레톤·봅슬레이·루지 엘리트 선수, IRB 준수, 비식별화 |
| **데이터 수집** | 평창 주행기록 + 선수 신체 데이터 + 기상청(KMA) API 병합 |
| **전처리** | 4단계(Phase 0~3) 이상치 자동 필터링 (제빙 상태, 충돌, crash 반영) |
| **통계 분석** | Python statsmodels, 다중선형회귀, VIF 다중공선성 통제, t-검정, p-value |
| **시스템 구현** | FastAPI 백엔드 + 실시간 환경 데이터 연동 웹 기반 AI 예측 대시보드 |

### 2.6 통계 분석 절차

```
1. 기술통계    →  평균, 표준편차, 최댓값, 최솟값
2. 상관분석    →  피어슨 상관계수 + 유의확률(p)
3. 다중선형회귀 →  R², Adjusted R²
4. 오차 검증   →  MAE, RMSE
5. 다중공선성  →  VIF
6. 유의성 검증 →  t-검정, p-value
```

## 4. 선행연구 vs 실측 데이터 비교

### 4.1 Start→Finish 증폭 효과

| 출처 | 증폭 배수 | 비고 |
|------|:---------:|------|
| **선행연구** (Vracas, 국제대회) | **1.5~2.0×** | 0.1초→0.15~0.2초 |
| **본 연구 실측** (평창, 남자) | **3.84×** | 0.1초→0.384초 |
| **본 연구 실측** (평창, 여자) | **2.34×** | 0.1초→0.234초 |

> **우리 데이터의 증폭 효과가 선행연구의 약 2배**
> - 가능한 원인: (1) 평창 트랙 특성(경사, 길이), (2) 훈련 데이터 vs 대회 데이터 차이, (3) 선수 수준 차이(국가대표급 vs 신인·유소년 혼합)

### 4.2 얼음 온도의 이론 vs 실측 충돌

| 관점 | 내용 |
|------|------|
| **선행연구 이론** (Poirier F.A.S.T.) | 얼음 온도가 마찰에 비선형 영향, 최적 온도대 존재 (-10~-20°C) |
| **프로포절 설계** | 얼음 온도를 주요 환경 독립변수로 설정 |
| **실측 분석 결과** | r=0.08, 편상관 0.03 — **통계적 무의미** |
| **원인 분석** | 평창 알펜시아는 **실내 냉각 시스템**이 빙면 온도를 일정하게 유지 → 변동이 극히 작아 변인으로서의 분산이 없음 |

> 이것은 논문에서 **"실내 냉각 트랙에서는 빙면 온도의 예측 기여가 무시 가능"**이라는 중요한 발견으로 기술할 수 있음.
> 선행연구(Poirier)의 마찰 모델이 옥외 또는 온도 변동이 큰 환경을 전제한 것이므로, 냉각 시스템이 잘 관리되는 현대 실내 트랙에서는 결론이 달라진다는 점이 학술적 기여.

### 4.3 기상청 API 연동의 실효성

```
인과 사슬:  외기온/습도/기압  →  공기밀도  →  항력  →  Finish
